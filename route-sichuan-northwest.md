## 路线总览

```
郑州 → 宝鸡 → 太白山 → 九寨沟 → 黄龙 → 都江堰/青城山

```

## ⚠️ 注意事项

- **高海拔**：黄龙 3500m+，九寨沟 3000m+，登山节奏建议比低海拔放慢 30%
- **住宿**：九寨沟暑期住宿紧张，需提前 1-2 周预订
- **车辆**：全程铺装路面，轿车无压力；九寨沟山路弯多但柏油完好
- **装备**：即使是七月，峨眉金顶/太白山顶清晨可能接近 0℃，备轻薄羽绒服
- **预算参考**：经济型 7–9k/人（含油费、住宿、门票、餐饮）
