# 🥇 川西北·九寨黄龙+峨眉登山美食线

> **推荐等级**：首选 | **总里程**：约 3,534km（往返全程） | **建议天数**：15天 | **5A景区**：6个  
> **出发地点**：郑州中牟·龙苑居住区 | **车辆**：轿车 ✅ | **七月气温**：山岳段 8–22℃ / 城市段 28–33℃  
> **数据来源**：高德地图 API 实时测算  
> **设计原则**：单日驾驶 ≤7.5h（仅 D4 达 7.5h）· 不走回头路

---

## 路线总览

```
龙苑居住区 → 宝鸡 → 太白山 → 九寨沟 → 黄龙
    → 都江堰/青城山 → 峨眉山 → 乐山大佛 → 成都
    → 汉中 → 三门峡 → 龙苑居住区
```

## ⚠️ 注意事项

- **高海拔**：黄龙 3500m+，九寨沟 3000m+，登山节奏放慢 30%
- **住宿**：九寨沟暑期紧张，提前 1-2 周预订
- **车辆**：全程铺装路面，轿车无压力
- **装备**：七月峨眉金顶/太白山顶清晨可能接近 0℃，备轻薄羽绒服
- **预算参考**：经济型 7–9k/人（含油费、住宿、门票、餐饮）
